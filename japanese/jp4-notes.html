N1 & N2 语法查缺补漏

发表于 2026-07-03 分类于 日语 本文字数： 11k 阅读时长 ≈ 35 分钟

考前再复习一下不熟悉的语法~

AあってのB（有A才B）

  • 国民あっての国家
  • 理性あっての人間
  • 実験の成功は皆さんの努力あってのことだ

Aいかんで / いかんによって + B（取决于A）

  • = A次第で、B…
  • A不同，B也跟着变化（いかん = 如何）
  • 結果いかんで対応を決める
  • 大統領の発言いかんで、株価は上がったり下がったりする

Aいかんでは / いかんによっては + B（取决于A，甚至会出现B）

  • B = 具体的な一つの状況
  • 在某些情况下，甚至会出现B这种结果
  • 話し方いかんで相手に与える印象は変わる
  • 話し方いかんでは相手に不快感を与えることもある

Aいかんによらず・Aいかんにかかわらず + B（无论A如何，都B）

  • Aがどうあっても関係なくBだ
  • 能力や成果のいかんによって、報酬額が変わります
  • 能力や成果のいかんによらず、報酬額は一定です
  • 犯罪は理由のいかんにかかわらず絶対してはいけない

A意向形 + が/とも + B（就算A也…；不管A也…）

  • 〜ても関係なく〜だ
  • 何を言われようが、考えを改めるつもりはない
  • たとえ両親に反対されようとも、私はこの人と結婚すると決めた
  • 僕の大切な妹をいじめる奴は誰だろうと許さない
  • 注：い形容詞也可以。比如 おいしい → おいしかろうが

V意 + が/と + V辞 + まいが/と（不管做不做A，都B）

  • AしてもAしなくてもBだ
  • 食べ放題では、たくさん食べようが食べまいが料金は同じだ
  • 雨が降ろうが降るまいが、イベントは行われるそうだ
  • 食べようが食べまいが = …食べるまいが
  • こようがくるまいが = …こまいが
  • しようがするまいが = …しまいが = …すまいが

V意 + にも + V可能ない（即使想…也不能/没办法…）

  • 暑くて寝ようにも寝られない
  • 財布を忘れて、買おうにも買えなかった
  • = 買おうにも財布を忘れて買えなかった

限りだ = とても〜だ（非常A）

  • N1に合格した！うれしい限りだ！

Aたら最後 / たが最後 + B（一旦做A就B）

  • B通常是坏事
  • 〜したら、〜のような悪い結果になる
  • カンニングなんてしたら最後、退学になる
  • 失敗したら最後、首にされてしまう
  • 彼にお金を貸したが最後、絶対に返してもらえないよ

N + かたがた（A顺便B）

  • = Aを兼ねてBをする
  • 結婚の報告かたがた、実家に帰ることにした
  • 東京出張かたがた、スカイツリーを見に行ってきた
  • 注：A基本都是 挨拶、報告、お見舞い、お礼、お詫び

V辞 + かたわら = ながら

  • 彼女は大学に通うかたわら、モデルとして活動している

がてら = のついでに / をかねて

  • 勉強がてら音楽をきく
  • 買い物がてら友達に本を返しに行った

V辞 + が早いか + B（刚一做A就B）

  • 〜すると、すぐ…した
  • 只能用在过去，不能用在未来
  • 家に帰ってくるが早いか、また出かけていった
  • 大学を卒業するが早いか、日本に来ました

数量詞 + からする / からある / からの（A以上）

  • 彼氏が１００万円からする指輪をくれた
  • 彼女は１０キロからある荷物を持ち上げることができます
  • 昨日の事故で、１００人からの人がケガをした
  • からする - 値段
  • からある - 数量・高さ・大きさ
  • からの - 人数

きらいがある（有…的倾向）

  • 只能用在负面的倾向
  • 这里的「嫌い」是名词用法
  • 他人を見下すきらいがある
  • 今の子供は野菜をあまり食べないきらいがある

〜極まりない・〜極まる = 非常に〜だ

  • 残念極まりない
  • 彼女の生活は、不健康極まりない
  • = 彼女の生活は、不健康なこと極まりない
  • = 彼女の生活は、不健康極まる

〜ごとし・〜ごとく・〜ごとき = まるで〜のようだ

  • かわいいこと天使のごとし
  • 天使のごとくかわいい
  • 天使のごときかわいさだ

AこととてB = なので（因为A，所以B）

  • 慣れないこととて失敗してしまいました
  • １０年前のこととて、何も覚えていません…
  • 新人のこととて、わからないことも多いと思います
  • 注：一般B都是谢罪和道歉

N + なしに・V辞 + ことなしに（没有…的状态下,就…）

  • 連絡なしに来られたら迷惑だよね
  • 今日は朝食を食べることなしに学校へ行った
  • 努力することなしにN１に合格はできない

〜ずくめ = 〜ばかりだ

  • 无论好坏都能使用
  • 最近残業ずくめでとても忙しい

～ないではおかない・～ずにはおかない（不能不，绝对）

  • 【意志】絶対に〜してやる
    • 仕返しをしないではおかない
    • 彼を処罰せずにはおかない
  • 【自発】自然に〜させてしまう
    • 彼氏のうそは、彼女を不安にさせないではおかなかった
    • あの映画は、みんなを感動させないではおかなかった

ずには済まない・ないでは済まない（非…不可）

  • 弁償しないでは済まない
  • 君が悪いのなら謝らないではすまない
  • 検査結果によっては手術せずにはすまなくなるだろう

すら = さえ

  • お腹を壊したので、おかゆすら食べられない
  • アメリカに留学したことがあるのに、日常会話すらできない

V辞/た形 + そばから（刚…就…）

  • するとすぐに
  • 新しい単語を覚えたそばから忘れてしまう
  • 痩せたいと言ったそばから焼きそばとラーメンを食べている

たところで

  • 【その時】ご飯ができたところで夫が帰ってきた
  • 【たとえ～でも】今さら謝ったところで許されないだろう

だに（就连…；只要…就…）

  • Aさえしない / ～するだけで
  • 微動だにしない = 全く動かない
  • 考えるだに恐ろしい事態だ
  • 今の人には想像だにできないだろうね

たりとも = １〜も〜ない（就连…也不能，哪怕…也不）
一秒たりとも寝ていない
試合中は１分たりとも油断するな！
= 試合中は１分も油断するな

Nたる + N（作为…，应该）

  • 后面一般接这个身份地位的规范、职责
  • 政治家たる者…
  • 彼には政治家たる資質がない
  • 医者たるもの、患者を第一に考えるべきだ

V連 + つ + V連 + つ（动作交替进行）

  • = 〜たり〜たり
  • 一组动词，可以互为反义，也可以是主动被动
  • 雲の間から、月が見えつ隠れつしている
  • 毎朝、押しつ押されつ満員電車に乗っている
  • 昨日の野球の試合は、抜きつ抜かれつでおもしろかった

Aであれ(Bであれ)C（不论…还是…，都…）

  • であれ = であろうと = であろうが
  • 只用一个「であれ」的时候，通常会搭配疑问词
  • 理由は何であれ、遅刻した者の途中入室は認めません
  • 電車であれ飛行機であれ、この天気では無理だと思うよ
  • 大人であれ子供であれ、人に迷惑をかけてはいけない
  • 晴れであろうと雨であろうと、イベントは予定通り行います

Vて + からというもの（自从…之后）

  • 〜てから / 〜て以来
  • をきっかけに、状態が大きく変わった
  • 后面不能接单次的动作，只能是持续的状态
  • 妹が生まれてからというもの、お母さんは忙しそうだ
  • 結婚してからというもの、お酒を飲んでいない
  • 一人暮らしをしてからというもの、カップラーメンを食べてばかりいる

でなくて何だろう（不是…又是什么呢）

  • これが悲劇でなくて何だろう

Aじゃあるまいし（又不是…）

  • = 〜じゃないだろう
  • 子供じゃあるまいし、自分で買いなさい
  • 金持ちではあるまいし、そんなに高いものは買えないよ！

〜てやまない（…不已；一直…）

  • ずっと〜ている / すごく〜ている
  • Aの感情や動作がずっと強く続いている
  • 愛してやまない妹
  • 彼は私が尊敬してやまない先生だ
  • 二人の幸せを願ってやみません

〜とあって（因为）

  • という状況なので（Bという状況も当然だ）
  • 后面不能接推量/可能
  • ラーメン１杯１００円とあって、３０分で完売した
  • ゴールデンウィークとあって、人気カフェにはたくさんの人が並んでいる

〜とあれば・〜とあらば = ～なら

  • あらば = 古い条件形
  • 常用搭配：〜のためとあれば
  • 山田くんは、お金のためとあれば何でもする
  • かわいくて性格もいいとあれば、モテて当然だよ

NといいNといい（…也好…也好）

  • 色といいデザインといい、とてもステキだ

といったところだ / というところだ（大概是…；差不多…那个程度）

  • 口语形：ってとこだ
  • だいたい〜くらいだ / 〜という感じだ
  • 発車まで５分といったところだ
  • あの車は３００万円といったところだろう
  • あと４０分くらいで終わるといったところだな

Aといえども（即使；虽然）

  • たとえAでも / いくらAでも
  • Aの状態でも、予想することと違う場合もある
  • 先生といえども、間違えることはある【地位】
  • プロといえども失敗することはある【地位】
  • ９月といえども、まだまだ暑い日もある【予想】
  • 体にいいといえども、食べすぎるのはよくない【予想】

といったらない = とても〜だ（…极了）

  • 由来：といったら(他に比べるものが)ない
  • 北海道の寒さといったらない
  • 北海道の寒さったらない
  • 北海道は寒いといったらない
  • 北海道は寒いといったらありはしない
  • 北海道は寒いといったらありゃしない
  • = 北海道はとても寒い
  • お母さんの料理はおいしいといったらない

〜(か)と思いきや（原以为…没想到…）

  • 「思いきや」的「き」是古语的过去形
  • 所以后面必须是过去形
  • 勉強しているかと思いきや、寝ていた
  • 行ったと思いきや、また戻ってきた
  • 今日はずっといい天気だと思いきや、急に雨が降ってきた

〜ときたら（说到，提到）

  • うちの息子ときたら、勉強もしないでゲームをしてばかりいる
  • 唐揚げときたら、やっぱりビールだね！
  • 金曜日の夜ときたら、一杯飲まずにはいられない

〜ところを（正…的时候；…之时）

  • Aという状況のときにBする
  • お忙しいところをすみませんが、この文法を教えていただけますか？
  • お疲れのところを悪いんだけど、これに目を通してくれる？

にしたところで / としたところで（即使…也…；就算…也…）

  • Aの立場から考えてもBだ
  • したところで = したって
  • 后面接悲观的内容
  • 進学しようとしたって学力がないとできない
  • 整形しようとしたところで中身まで変わるわけではない
  • どれだけおいしいにしたって、食べ過ぎると飽きる

〜とは = なんて（竟然；没想到）

  • N1に満点で合格するとは驚いたわ！
  • まさかあの人が失敗するとは…
  • 注：「とは」作为定义说明用法时，是「というのは」的省略

〜とはいえ = けれども（虽然…但是…）

  • 飲み放題とはいえ、飲みすぎはよくないよ…
  • 英語が話せるとはいえ、日常会話できるくらいだ

〜と言わんばかりに・〜とばかりに（好像在说…，像是…）

  • うちの犬は、私が帰ると「おかえり」と言わんばかりに走ってくる
  • 彼は、こんな問題は簡単だとばかりに、どんどん問題を解いていく
  • あの人に恋人が居ないことを知り、ここぞとばかりに告白した
  • まずいとばかりの顔をして食べている
  • 注：ここぞとばかりに = 机会来啦，趁千载难逢的好机会

〜ともなく・〜ともなしに（前接疑问词 + 不知…）

  • （不確か）〜か、よくわからないが〜
  • いつ + 助词 + ともなく - 不知何时
  • だれ + 助词 + ともなく - 不知是谁
  • どこ + 助词 + ともなく - 不知从哪
  • 山田くんは会計をする時になると、どこへともなく行ってしまう
  • どこからともなく、パンのいい匂いがする

〜ともなく・〜ともなしに（漫不经心地）

  • 動A［る］ともなく動A［たら］
  • 動A［る］ともなく動A［る］と
  • （無意識）〜しようというつもりなく、〜する
  • テレビを見るともなく見ていたら、…
  • 夜空を見るともなしに見ていたら、流れ星が見えた
  • ラジオを聞くともなく聞いているうちに寝てしまった

〜ともなると・〜ともなれば（一旦…的话；要是…）

  • Aという状況/場合になると、当然Bという状況になる
  • ３連休ともなると高速は渋滞する
  • ケーキが１つ無料ともなると、多くのお客さんが来るだろう
  • 海外旅行ともなれば、しっかりバイトして貯金しなきゃいけないな…

～ないまでも（没有…至少也…；即使做不到…也该…）

  • 高层次的达不到，但是低层次的还是可以的
  • 予習や復習はしないまでも、宿題くらいしなさい
  • １００点とは言わないまでも、８０点は取りたい
  • １０Kgとは言わないまでも、３Kgはやせたいな…
  • 毎日とは言わないまでも、せめて週に一度は電話してきてほしい
  • 彼女は美人だとは言わないまでも、かわいいと思うよ
  • 常见搭配：〜とは言わないまでも

～ないものでもない・～ないでもない・～なくもない（也并非不）

  • = 少し〜の可能性がある
  • すごく高いが、買えないものでもない
  • 彼の言いたいことがわからないものでもない
  • 今から急いで行ったら、間に合わないものでもない
  • 「ないものでもない」一般用于个人判断和意向
  • 「ないこともない」还能用于客观事实

～ながらに・～ながらの（保持…的状态）

  • 一般只有这几个搭配：
    • 生まれながら（天生）
    • 昔ながら（一如既往）
    • 涙ながら（流着泪的状态）
  • 昔ながらの味
  • 昔ながらの伝統を大切にしよう
  • 彼女は辛い経験を涙ながらに話した
  • 彼は生まれながらの天才です
  • 人には必ず生まれながらの使命がある

Nなくして(は) = Nがなかったらダメだ（如果没有…就不能…）

  • 愛情なくして夫婦は務まらない
  • 失敗なくしては成功もありません
  • 正当な理由のあるときを除き、当該個人の同意なくして、個人情報を第三者へ開示いたしません

N + ならでは(のN)（只有～才有的～）

  • 納豆は日本ならではの食べ物だ
  • 日本ならではの観光地へ行ってみたい
  • この文化はこの国ならではのものだ

V辞 + なり = 〜すると、すぐに〜（一…就立即…）

  • 后面接马上发生了的事情，带有意外性
  • 所以不能接未来、意志、希望、要求
  • 也不能用在自己身上，前后主语也要一致
  • 娘は帰ってくるなりゲームをし始めた
  • 彼女は手紙を読むなり笑い出した

〜なり〜なり（要么…要么…）

  • 带有限制的「や～など～」「たり～たりする」
  • 一般只用于：命令、请求、建议、忠告
  • 煮るなり焼くなり好きにしろ
  • パンなりおにぎりなり食べて行きなさい
  • 掃除なり洗濯なり手伝ってちょうだい
  • コーヒーなりジュースなり、好きなものを飲んでください

なりに / なりの + N（符合…程度，与…相应的程度、方式）

  • 〜にふさわしい程度の
  • 私［自分］なりに - 自谦说法
  • 私なりに色々考えた企画です。ぜひご検討ください
  • 料理が下手なりに頑張って作りました
  • 安い物には安いなりの理由があるはずだ
  • 才能がないなら、ないなりに生きていく
  • 課長、先日、提出した案はどうお思いになりますか。あくまで、私なりの考えなんですが…

～に(は)あたらない = Aする必要/価値はない（不必；不值得）

  • 一度失敗したぐらいで悲観するにはあたらない
  • ギリギリで合格したから、喜ぶにはあたらないよ…

～てはたまらない・～てはかなわない = 〜ては困る（让人受不了）

  • 試験に遅れてはかなわないので、今日は早く寝よう
  • 忙しいからといって、毎日残業させられてはかなわない
  • てはたまらない：只能用于负面
  • てたまらない：正面负面皆可
  • 例：嬉しくてたまらない

N + にあって（处于…情况下）

  • どんな困難な状況にあっても、解決策は必ずある
  • あの会社は経営状況の悪化にあって、倒産を余儀なくされたそうだ

〜に至って（到了…阶段）

  • 友達に指摘されるに至って、間違った日本語を使っていたことに気づいた
  • 上司に注意されるに至って、敬語の使い方がおかしいとわかった
  • ストレスで病気になるという状況に至って、会社を辞めることを決意した

〜に至るまで = 〜までも（甚至到）

  • あの映画は子供から大人に至るまで、みんなに人気だ
  • お母さんは和食はもちろん、イタリアンから中華料理に至るまで何でも作れる

～にかかわる = ～に関係する（关系到…；影响到…）

  • 会社の存続にかかわる
  • 命にかかわる重病

〜にかたくない（不难）

  • = 〜するのは難しくない
  • 彼女の悔しさは想像にかたくない
  • 彼の気持ちは理解するにかたくない

〜にして（段階；並列）

  • 段階 = 到了…阶段
  • 並列 = 是…同时也是…
  • ３回目にして、やっとN１に合格することができた
  • あの子は９歳にして大学に合格したそうだ
  • 彼女は女優にして、３児の母である
  • 彼はサラリーマンにして世界的に有名な画家でもある

〜に即して（按，根据）

  • 現実に即して行動する
  • 時代に即した考え方をするべきだ

〜にたえる・〜にたえない（值得…；不值得…）

  • 〜する価値がある/ない
  • 鑑賞にたえる絵
  • 鑑賞にたえない絵
  • 鑑賞にたえる絵ではない
  • そんな話は聞くに堪えない
  • この本は子供向けだが、大人でも読むにたえる

〜にたえない（…不尽）

  • いつもお世話になり、感謝にたえません
  • 色々とご支援くださり、感謝にたえません

V辞に足る = Vできる（足以）

  • 旅行するに足るお金が貯まった
  • 満足するに足る成績がとれた
  • 満足に足る成績がとれた

AにひきかえB（与…相比 正相反）

  • 是「引き換える」的连用形
  • 姉に引き換え、妹はお転婆だ
  • 昨日にひきかえ、今日は寒い
  • 昨日までの寒さにひきかえ、今日はとても暖かい
  • 友達は結婚していくのにひきかえ、私は彼氏すらいない

N/疑問詞 + にもまして（比…更加…）

  • Aもそうだが、Aはそれ以上にB
  • 今回のN１は、前にもまして難しかった
  • 今日の山田くんは、いつにもましてケチだ
  • いつにもまして → いつも以上に
  • 誰にもまして → 誰よりも
  • 何にもまして → 何よりも一番・最も
  • どこにもまして → どこよりも
  • 前にもまして・以前にもまして → 前よりもっと

〜の至り・〜の極み = 非常に〜だ（极其，至极）

  • 感激の至り
  • 光栄の至り
  • 何から何までお世話になり、恐縮の至りです
  • 彼女は贅沢の極みを尽くした生活をしている
  • １点足りず不合格だったとは、痛恨の極みだ

Vの / N + はおろか（不用说；岂止是A，连B都）

  • = 〜はもちろん、〜も
  • 只能接惊讶，不满等负面内容
  • 「〜はもちろん」正面负面皆可
  • 金はおろか命まで奪う
  • 酒はおろかジュースさえ飲まなかった
  • 掃除はおろか、布団を上げたこともない
  • このアパートにはお風呂はおろか、トイレもない
  • 足の骨が折れてしまったので、歩くのはおろか立つことさえできない

AばこそB = AだからこそB（正因为…才…）

  • 動ば形 + こそ
  • イ形語幹 + ければこそ
  • ナ形語幹 + であればこそ／ならばこそ
  • どんなことも命あればこそできる
  • 死を思えばこそ、生の意味を知る
  • 健康であればこそ、楽しい時間を過ごせる

〜たらそれまでだ・〜ばそれまでだ（若…就结束了；…的话也就没有意义了）

  • = それで終わりだ
  • 先生に見つかったらそれまでだ
  • たくさんお金を貯めても、死んだらそれまでだよ

V辞 + べからず / V辞 + べからざる + N（禁止）

  • 「べからず」是「べし」的否定形
  • 絵に触れるべからず
  • 働かざる者、食うべからず
  • エサを与えるべからず
  • 半導体は電子部品に欠くべからざる物質です

～のではあるまいか = ～のではありませんか

  • 慢性的な睡眠不足が体に様々な悪影響を及ぼしているのではあるまいか

V辞 + べく = AするためにB（为了）

  • 日本語を勉強すべく、日本に来ました
  • 車を買うべく、貯金をしている

〜まじき（不该…的）

  • 基本只用在「あるまじき」「許すまじき」
  • カンニングは、あるまじき行為だ
  • 万引きをするなんて、警察官としてあるまじき行為だ
  • 飲酒運転は許すまじき行為だ
  • 「べし」「まじ」是古语的一组反义词

V辞 + までだ・〜までのことだ（大不了就是…而已）

  • = （〜がだめなら）〜すればいいから、問題ない
  • 電車がなくなったら、タクシーで帰るまでだ
  • お金が足りなかったら、お母さんに借りるまでだ

Vた + までだ・〜までのことだ（不过是…）

  • = ただ～しただけだ
  • 私は事実を述べたまでのことだ
  • 聞かれたから答えたまでだ

までもない / までのことはない（不必…，用不着…）

  • = 〜する必要がない
  • 言うまでもない
  • 電卓を使うまでもないさ
  • 風邪を引いただけだから、病院に行くまでもないよ
  • 簡単な文法ですから、説明するまでもないですね
  • そんなに遠くないし、タクシーに乗るまでのことはない

さることながら（不用说，不言而喻）

  • = AはもちろんB
  • お金もさることながら、命はもっと大切だ
  • その顔の美しさもさることながら、何より声が素敵
  • あのレストランは、料理もさることながらサービスがすばらしい
  • ここの料理はそのボリュームもさることながら、視覚的な満足感もすごい

Aば/たら + BものをC = 〜のに（要是…就好了）

  • 言ってくれたら、助けてあげたものを、なぜ黙っていたんですか
  • 電話をしてくれたら、持っていってあげたものを…
  • 教えてくれたら、手伝いに行ったものを…

や否や = 〜するとすぐに（一…立刻就…）
えびを食べるや否や、体が痒くなった
薬を飲むや否や、腹痛が治った
彼女は、家に帰るや否やゲームを始めた

V/A/N + (が)ゆえに（因为...所以）

  • AだからB。古文。硬い。
  • 彼女が美しいが故に、人生がうまく行ってる
  • 彼は失恋を恐れるゆえに、告白できないでいる
  • 山田くんはケチゆえにモテないんだと思う

～をおいて～ない（除了…之外没有…）

  • 貯金上手といえば、ぼくをおいてほかにはいないと思うんだ〜
  • 結婚するなら、彼をおいてほかには考えられないわ
  • 美人でダンスが上手な歌手は、彼女をおいてほかにはいないだろう

～を限りに・～限りで = を最後に（以…为最后；以…为止）

  • 今日を限りにお酒はやめよう！
  • 健康のために、今日を限りにタバコは吸わないぞ
  • 今回を限りに二度と遅刻をしません
  • この話はこの場限りで忘れてね
  • 子供のころよく遊んだ遊園地は今月限りで閉鎖されることになった

N + をもって（以…，凭…）

  • 「もって」的汉字形式是「以て」
  • 用于表达<手段・媒介>的场景
  • 書面をもってお知らせします
  • 面接結果は、メールをもって通知します
  • 拍手をもってお二人をお迎えしましょう
  • イベントを行うかどうかは、電話をもって連絡します

～を皮切りにして / を皮切りとして / を振り出しに（开始，以…起手）

  • 接下来会有同类动作陆续发生
  • 東京公演を皮切りに日本各地で演奏会を開く
  • 東野圭吾を皮切りに、いろいろな小説を読むようになった

N + を禁じ得ない（不禁…）

  • 涙［喜び］を禁じえない
  • この社会に絶望を禁じ得ない

〜をものともせずに（不以为意，不把…当回事）
お母さんは台風をものともせずに買い物に行った
周囲の反対を物ともせず、ひたすら意志を通した
子供たちは寒さをものともせずに、外で遊んでいる

〜を余儀なくされる・〜を余儀なくさせる（被迫，不得已做某事）

  • = やむを得ない = 仕方ない
  • 辞職を余儀なくされる
  • 撤退を余儀なくされた
  • XはAを余儀なくされた（受身形）
  • = XはAするしかない状況に追い込まれた
  • YがXにAを余儀なくさせた（使役形）
  • = YがXをAするしかない状況に追い込む
  • X：人・組織
  • Y：事件・事情・原因
  • 慢性的な人手不足で社員は超過勤務を余儀なくされている
  • 慢性的な人手不足が社員に超過勤務を余儀なくさせている
  • コロナの蔓延で、多くの飲食店が休業を余儀なくされた。
  • コロナの蔓延は、多くの飲食店に休業を余儀なくさせた。

Nをよそに（丢在一边，不顾不管）

  • よそ = 余所
  • 〜を無視して
  • 〜を気にしないで
  • 〜に関係なく
  • 勉強をよそに遊ぶ
  • 彼は、お母さんの期待をよそに会社を辞めてしまった

Aんがために = AするためにB（为了…）

  • A（目的）必须要是重要，特别的
  • 普通的目的（比如去超市），直接用「するために」
  • N１に合格せんがために、毎日がんばって勉強した
  • 大金を得んがために、犯罪を犯した
  • 注：这里的「が」就是现在的「の」，比如「我が社」「自由が丘」

現代・古文 意向形

  • 飲もう - 飲まん
  • 受けよう - 受けん
  • 来よう - 来ん（こん）
  • しよう - せん

V未 + んばかりに（几乎快要/就要…，几乎显出…的样子）

  • スル → せんばかりに
  • 今にも～しそうなくらい～
  • 溢れんばかりの性欲
  • 彼女の溢れんばかりの笑顔に癒された
  • 頭が割れんばかりの頭痛に悩んでいる
  • 土下座せんばかりに必死に謝ったが許してくれなかった

〜次第だ = 〜というわけだ（理由）

  • 事故で電車が止まり、遅刻した次第です
  • 母が病気になったため、帰国することにした次第です
  • 日常生活见不到，商业场合比较多

V辞 + に + V可能否定形（想做…却不能…）

  • 〜たいのに〜できない
  • 蚊がいたので寝るに寝られなかった
  • お金がないので、買うに買えなかった
  • 上司が残業しているので、帰るに帰れなかった
  • 受験に失敗した彼はとても落ち込んでいて、これからどうするかなんて今は聞くに聞けない

AたらAたで（如果真的A了，那么A自然又有A自己的……）

  • すごく眠いのに寝たら寝たですぐ起きてしまう = 终于睡着了，可睡着以后又马上醒了
  • 初恋の人に会ってみたいけど会ったら会ったで気まずいだろう = 真见面了，又会尴尬
  • 最初はあまり興味なかったけど、観たら観たで面白かった = 真看了以后，又发现有意思
  • お金持ちになるのは簡単じゃないし、お金持ちになったらなったでお金持ちなりの苦労も待っている = 有钱了，自然有有钱人的烦恼

AならAで（如果…就…）

  • 「な形容詞・名詞」的情况，更倾向用「〜なら〜で」
  • 反対なら反対で、ちゃんと理由を説明しなさい
  • = 如果反对的话，就拿出反对的理由
  • 嫌なら嫌で、そう言えばよかったじゃない！
  • 嫌だったら嫌だったで、そう言えばよかったじゃない！
  • = 如果你不喜欢，那就直接说不喜欢啊！

に越したことはない（~再好不过了）

  • 給料は多いに越したことはない
  • 健康のためには、タバコは吸わないに越したことはない
  • 住む場所は静かであるに越したことはない
  • 注：な形・名詞は「ーである＋〜」という言い方もある

V辞 + べくして + Vた（结果完全是意料之中）

  • 当然の結果として〜になった
  • 彼女はN１に合格するべくして合格した
  • 彼は毎日がんばって練習していた。試合には勝つべくして勝ったんだよ
  • 「べし」表示：应该、理所当然、必然
  • 「VるべくしてVた」字面就是：该V，所以V了

AならまだしもB（…的话就算了；…的话还好）

  • 〜だったらまだいいが、〜は
  • ３０分くらいならまだしも、２時間も並ぶなんてバカだね
  • １回ならまだしも、２回も失敗してしまった
  • 逃げるだけならまだしも、俺は嘘を吐いてお前の父親に濡れ衣まで着せたんだ。許せないだろ？

～のをいいことに（利用…；以…为借口）

  • Aの状況、機会を利用してB(よくないこと)をする
  • 休みなのをいいことに昼まで寝てた
  • 顔見えないのをいいことにひどいこと言う人は卑怯だ
  • 研究室に私しかいないのをいいことに、音楽をスピーカーで流しながら仕事をする

～た弾みに／弾みで／た拍子に／拍子で（刚一…就…）

  • 転んだ弾みに、たまごを割ってしまった
  • 重いものを持ち上げたはずみに、腰を痛めてしまった
  • 友達にぶつかった拍子で、コーヒーをこぼしてしまった

～わ～わで = 〜し〜し（又…又…）

  • 傘を忘れるわ、スマホがないわで大変だった
  • = 傘を忘れるし、スマホがないし大変だった
  • バイトがあるわ、テストが近いわで忙しい
  • 基本上都是用在负面情况，比如「大変だ」「困った」

〜だの〜だの = 〜や〜など（…啦…啦）

  • 只能用在：非難する気持ち・不満
  • 主人はバッグだの靴だの、娘が欲しがるものは何でも買ってくる…
  • お母さんは勉強しろだの早く寝ろだの、本当にうるさい
  • 彼は給料が安いだの仕事が忙しすぎるだの、文句ばかり言っている

ありき（以…为前提）

  • ～の存在を前提として
  • 結論ありきの会議に参加しても意味がない
  • 残業ありきでスケジュールを組むのは間違っている

AならいざしらずB（如果…的话另当别论，但是…的话就很严重了）

  • Aならまだいいが、Bは
  • 英語を始めて１年くらいならいざしらず、５年も勉強しているのに話せるようにならない
  • ２０年前ならいざしらず、現在はネットなしでは生活できないと思う
  • A：彼氏にドタキャンされたって本当？
  • B：うん。仕事ならいざしらず、友達と飲みに行くからって言われたの…
  • A：ひどい！そんな人とは別れたほうがいいよ！！

～ようによっては／ようでは（取决于/根据…的方式）

  • ようによって = いろいろな状況
  • ようによっては = 具体的な一つの状況
  • 言葉は言いようによって人を喜ばせも悲しませもする
  • この絵は見ようによって少女に見たり老婆に見えたりする
  • この選手は癖が強いが、使いようによってはチームの中でうまく機能するはずだ
  • ここから目的地まで、道の混みようによっては１時間かかる場合もある

AひとつでB（全凭…）
話し方ひとつで、相手にいい印象を与えることができるものです

Vた + ためしがない（从来没有…）

  • 今まで〜だったことがない
  • 只用于负面的情况（非難する気持ち）
  • 彼女はいつも三日坊主で何をやっても続いたためしがない
  • よく宝くじを買うが、当たったためしがない
  • 彼は約束を守ったためしがない
  • 注：「ためし」写成汉字是「例」，意思是先例、前例

  • 田中さんは仕事が終わると、すぐ帰ってしまう。
